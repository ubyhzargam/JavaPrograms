This code takes the number of elements in the array, the array elements in sorted order and the element to be searched as input from the user in the main functions which then calls another function BS which is recursive by passing array, the key to be searched, low index and high index as parameters. This function implements Binary search through recursion and returns the main function the index of the element in the array if present else it returns -1. The main function prints the results. The code can be accessed from .java file in the folder.
